# Kth Largest Element

Finds the kth largest element in an unsorted array.

## Examples
Input: {24, 2, 45, 20, 56, 75, 2, 56, 99, 53, 12} and k = 1
Output: 99

Input: {3,2,1,5,6,4} and k = 2
Output: 5

# Usage
```sh
$ mvn clean install assembly:single
$ java -jar kthlargestelement-1.0-SNAPSHOT.jar
```
